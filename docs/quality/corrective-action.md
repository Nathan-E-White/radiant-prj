## Purpose

This procedure defines how findings, nonconformances, review comments, and improvement actions are captured and dispositioned.

## Finding Classes

| Class | Description | Closure Evidence |
| --- | --- | --- |
| Blocking | Prevents release or invalidates evidence | Fix plus passing verification |
| Major | Affects controlled behavior, requirements, or records | Fix or approved disposition |
| Minor | Editorial, clarity, or low-risk process gap | Review acknowledgement |
| Observation | Improvement opportunity | Optional follow-up |

## Required Fields

Findings shall identify ID, date, originator, affected item, class, description, disposition, owner, verification, and closure status.

## Disposition Options

- Correct now.
- Correct in a later baseline with documented rationale.
- Accept as-is with reviewer approval.
- Reject with documented basis.

## Closure

Closure requires evidence that the disposition was completed and that affected verification remains valid.
