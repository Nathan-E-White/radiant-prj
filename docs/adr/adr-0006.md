# Consolidate Workbench IA Into Status Workbench

| Field | Value |
| --- | --- |
| Document ID | ADR-0006 |
| Revision | 1.0 |

| Owner | Software |
| Baseline | Status Workbench refactor |

## Status

Accepted

## Context

The frontend had five top-level tabs: Kaleidos Brief, Compute Workbench, Simulator Workbench, Evidence Matrix, and SimOps Control. That split made related status work feel like separate products. The queue, run orchestration, simulation status panels, value-basis workbench, and evidence handoff all described one operational surface but were scattered across multiple tabs.

ADR-0005 already requires measured, simulated, imputed, and operational telemetry to stay distinct. The UI now needs to preserve that semantic separation while reducing top-level navigation.

## Decision

Use three top-level pages: Welcome, Status Workbench, and Evidence.

Status Workbench is the renamed user-facing workbench. It keeps the fleet strip, measured state, twin viewport, imputed state, simulated result state, and lineage rail as the top workbench region. It absorbs the compute queue and SimOps control plane as lower workbench regions. The four Simulation Ops visual panel concepts become an HPC Status Panel bay driven by the selected queue/job context.

The backend API and contract names may continue to use `simulator-workbench` where they are already part of the controlled implementation surface. The rename is a product IA and frontend surface decision, not a backend route migration.

## Consequences

Top-level navigation is smaller and the app reads as Welcome, Status Workbench, and Evidence. Compute Workbench and SimOps Control stop being standalone top-level surfaces. Value-basis labels remain explicit, and operational telemetry remains distinct from measured plant-like stand-ins, imputed twin state, and simulated result state.

The old Simulation Ops mock documents and workbench planning records must be read through this ADR: panel concepts belong in Status Workbench, not in a resurrected Compute Workbench tab.
