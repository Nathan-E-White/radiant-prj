## Teaser
* **🧪 Nuclear Context:** Real-time core telemetry for the 16 control drums and TRISO fuel cylinders.
* **🖥️ HPC Infrastructure Metrics:** AWS ParallelCluster dynamic auto-scaling state, AWS EFA (Elastic Fabric Adapter) packet drop rates, and spot instance cost tracking.
* **📊 Visual Elements:** A 2.5D schematic of a microreactor core where clicking a localized hot spot triggers a simulated "Cloud Bursting Active" network topology graph.

## Context
A panel that handles real-time 2.5D rendering of the microreactor’s core thermal mesh, capturing spatial kinetics across all 16 control drums and TRISO fuel passages. Under standard low-power standby or nominal conditions, a local hybrid cluster effortlessly processes these localized reduced-order simulator models. However, if a sudden core thermal transient or coolant channel blockage occurs, local compute nodes become instantly saturated. The system must immediately scale horizontally into AWS ParallelCluster using Elastic Fabric Adapters (EFA) to re-verify multi-dimensional safety margins without interrupting the active Hardware-in-the-Loop (HIL) telemetry client data loop.

## Decision
We will implement an automated, event-driven state machine within the Go backend that continuously evaluates core thermal metrics against hard-coded boundary constraints.

To prevent tracking logic from blocking network transport threads, boundary scanning is executed inline during the 20Hz serialization pass. If a thermal value exceeds a critical limit (e.g., >800°C), a thread-safe state variable (transient_active) is flipped to true. This flag is broadcasted upstream to the web layout over the persistent WebSocket link, instantly prompting the UI layer to switch to an alarm state and simulate an automated cluster expansion sequence (scaling from a 128-node local hybrid configuration up to a 1024-node AWS ParallelCluster spot instance topology via programmatic automation).

## Consequences
### Positive

#### Automated Elasticity
Shifts the burden of resource provisioning out of human hands, dropping compute scale-out latency down to sub-second software signal triggers.

#### Proactive UI Rendering
Toggling a simple boolean field over the socket payload allows the frontend WebGL layout to fluidly adjust its color spectrum maps and warning matrices without checking multiple heavy endpoints.

### Negative

#### Trigger Chattering Risk
Rapid sensor fluctuations hovering precisely around the 800°C threshold could trigger repeated allocation signals, requiring the eventual implementation of a software deadband or dampening hysteresis logic.
